# Genos
A Vue.js / Express.js web application for keeping track of guitar tabs
- built from codyseibert's instructional video, with added features

## Frontend
* Vue
* Vue Router
* Vuex
* Vuex Persistedstate
* Vuex Router Sync
* Vuetify
* Vue Youtube Embed
* Axios
* Lodash

## Backend
* Node Js
* Express
* Bcrypt Nodejs
* Bluebird
* Body Parser
* Cors
* Joi
* Jsonwebtoken
* Morgan
* Passport
* Passport-jwt
* Sequelize
* Sqlite3

# Setup

You need to have Node Js installed: https://nodejs.org/en/

You must at least have **NODE version 8.2.1**

Don't know how to install different version of node?  Try this: https://github.com/tj/n

### Client - Terminal A
```
cd client
npm install
npm start
```

### Server - Terminal B
```
cd server
npm install
npm start
```
